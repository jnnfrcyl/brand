Typography

Each font family chosen is quite diverse and covers a full range of typographical uses. Projects representing other entprises should not use the eMoney corporate font, so as to distinguish these communications from our own look and feel.

Kerning, letterspacing, and leading or linespacing should be done to enhance the look of projects. This is a matter of taste, but should be done with discretion.

Variations of the each font can be used for emphasis. Use bold, italic, or bulleted lists to emphasize specific information.

Web Typography

The Lato font family has many weights and should be given preference in all web communications aside from application fonts. Careful discretion may be used when selecting accent fonts and will be approved on a per-project basis.

Adhere to these general sizing guidelines:

  • h1: 36px
  • h2: 30px
  • h3: 24px
  • Body font: 16px regular
  • Small font: 14px

Use Bold font weight for headlines and Regular font weight for body text. Use text-light sparingly.

App Typography

Bariol Bold

Bariol Bold should be used only in the context of the software platform for headlines.

Adhere to these general sizing guidelines:

  • h1: 28px / 2em
  • h2: 25px
  • h3: 21px
  • h4: 18px
  • h5: 14px
  • h6: 12px

Helvetica Neue, Helvetica, Arial, sans-serif

The font stack for the software platform is

"Helvetica Neue", Helvetica, Arial, sans-serif;

  • The small selector is 85% of 14px, or about 11.9px.
